Setup Guide

1
Create Supabase Project Go to supabase.com and create a new project
2
Create memories table Run in SQL Editor: 
CREATE TABLE memories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  content TEXT NOT NULL,
  category TEXT NOT NULL,
  importance FLOAT DEFAULT 0.5,
  source_type TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  expires_at TIMESTAMP
);
3
Copy credentials Project Settings → API → URL and anon key

Memories are extracted from conversations every 10 minutes and stored securely in your Supabase database.
